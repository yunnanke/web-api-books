# Создание web API

## Описание задачи
Реализовать простейшие web API для личной тренеровки работы с контейнерами, эндпоинтами и тестированием.

## Техническая информация
Язык программирования: C# .NET 8.0
Среда разработки: VisualStudio  
Формат файла: `.cs` 

## Информация о разработчике
Фамилия: Матвеева
